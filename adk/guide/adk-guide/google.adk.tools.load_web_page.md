# Google ADK Tools Load Web Page Tool API Reference

`google.adk.tools.load_web_page` 모듈의 API 레퍼런스 문서입니다.

웹 브라우징을 위한 도구입니다.

## 함수

### load_web_page

```python
google.adk.tools.load_web_page.load_web_page(url)
```

URL의 콘텐츠를 가져와서 그 안의 텍스트를 반환합니다.

- **반환 타입**: `str`
- **매개변수**:
  - **url** (`str`) – 브라우징할 URL
- **반환값**: URL의 텍스트 콘텐츠
- **반환 타입**: `str`
