# Google ADK Tools Get User Choice Tool API Reference

`google.adk.tools.get_user_choice_tool` 모듈의 API 레퍼런스 문서입니다.

## 함수

### get_user_choice

```python
google.adk.tools.get_user_choice_tool.get_user_choice(options, tool_context)
```

사용자에게 옵션을 제공하고 하나를 선택하도록 요청합니다.

- **반환 타입**: `Optional[str]`

**매개변수:**
- **options** - 사용자에게 제공할 선택지들
- **tool_context** - 도구의 컨텍스트
